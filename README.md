# ecs-sdl

ECS pattern implementation for SDL2 library

## Game Of Balls

Implemented:
- [ ] Loading config from files
- [ ] Polygonal circles
- [x] EntityManager
- [x] Entity with components
- [x] Collision subsystem
- [ ] Score subsystem
- [x] Transform subsystem
- [x] Render subsystem
- [x] Spawn subsystem
- [x] Input subsystem
